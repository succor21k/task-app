작업지시서 PWA - 관리자/사용자 분리 버전

업로드할 파일:
- index.html
- admin.html
- user.html
- firebase-config.js
- manifest.json
- sw.js
- icon-192.png
- icon-512.png

GitHub 적용 방법:
1. GitHub 저장소(task-app)로 이동
2. Add file > Upload files 클릭
3. 이 폴더 안의 파일을 모두 드래그해서 업로드
4. 같은 이름 파일이 있으면 덮어쓰기 됨
5. Commit changes 클릭
6. 1~2분 뒤 새로고침
